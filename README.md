
# 📌 客制化外卖订单系统

基于 **Spring Boot + MyBatis + Redis + MySQL + Nginx + 阿里云 OSS** 开发的前后端分离客制化外卖订单系统，支持后台管理端与用户端，具备商品管理、订单管理、购物车、图片上传等核心功能。

---

## 🚀 项目简介

本项目为一个前后端分离的客制化外卖订单系统，包含：

* 👨‍💼 管理端（后台）
* 👤 用户端（小程序 / Web）
* 🖥 后端服务（RESTful API）
* ☁ 图片存储（阿里云 OSS）
* ⚡ 缓存优化（Redis）

项目目标：

* 熟练掌握 Spring Boot 企业级开发
* 理解事务管理与数据一致性
* 掌握缓存与性能优化
* 具备基础高并发设计思维

## 🔥 技术亮点
### 1️⃣ Redis 缓存优化

* 对菜品数据进行缓存
* 减少数据库压力
* 提高接口响应速度

---

### 2️⃣ 事务管理

* 下单过程使用事务
* 保证订单数据一致性
* 防止部分提交问题

---

### 3️⃣ 图片上传（阿里云 OSS）

* 使用 OSS 进行图片存储
* 后端返回访问路径
* 实现文件服务器与业务服务器分离

---

### 4️⃣ Nginx 反向代理

* 统一入口
* 解决跨域问题
* 提高静态资源访问效率

---

### 5️⃣ 工程化设计

* 分模块开发
* 统一返回结果封装
* 全局异常处理
* 统一日志规范


### 5️⃣ 引入JWT

* 解决HTTP协议无状态的缺点
* 实现用户一段时间内无需重复登录及身份验证功能
* 对Signature进行处理
* 增强重要信息的安全性。
